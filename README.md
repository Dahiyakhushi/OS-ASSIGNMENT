Assignment 3: Page Replacement Algorithms

Description

This assignment implements various Page Replacement Algorithms used in operating systems for efficient memory management.

Algorithms Implemented

* FIFO (First-In First-Out)
* LRU (Least Recently Used)
* Optimal
* MRU (Most Recently Used)
* Second Chance (Clock Algorithm)

Objective

* Understand virtual memory and paging
* Calculate page faults
* Compare performance of different algorithms

Input

* Number of Frames
* Page Reference String

Output

* Total Page Faults

Observations

* Optimal algorithm results in minimum page faults
* LRU performs better than FIFO
* Second Chance improves upon FIFO
* MRU is generally less efficient

Assignment 4: Disk Scheduling Algorithms

Description

This assignment focuses on disk scheduling techniques to reduce seek time and improve performance.

Algorithms Implemented

* FCFS (First Come First Serve)
* SSTF (Shortest Seek Time First)
* SCAN
* C-SCAN

Objective

* Understand disk scheduling concepts
* Calculate seek time
* Compare algorithm efficiency

Input

* Disk Requests
* Initial Head Position
* Disk Size

Output

* Total Seek Time

Observations

* SSTF gives the lowest seek time
* FCFS is the least efficient
* SCAN provides a balanced approach
* C-SCAN ensures fairness
